## アプリ名
### Hello Friend
## 概要
英語を学ぶ日本人と日本語を学ぶ外国人を繋ぐSNS。<br/>
タイムラインで学習内容のアウトプットや疑問点をシェアし、<br/>
コメントやいいねをくれたユーザーとチャット機能を使い親交を深める狙い。
## 開発のきっかけ
私自身が留学先で英語の学習中、ネイティブの友人にとても助けられたことから、<br/>
日本にいても気軽に助け合える友人に出会える場があればと考案。
